# gaycowboy.work

A personal blog and showcase celebrating queer rural life, labor, and identity.

## About

gaycowboy.work is a space for personal essays, reflections, and stories about navigating work, land, pride, and becoming in rural queer spaces. No corporate polish—just honest writing from lived experience.

## Built With

- **[Astro](https://astro.build)** – Fast, content-first static site generator
- **Markdown** – Simple, readable content format
- **Tailwind CSS** – Utility-first styling

## Getting Started

### Prerequisites

- Node.js 18+ 
- pnpm (recommended) or npm

### Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev
# Site runs at http://localhost:3000
